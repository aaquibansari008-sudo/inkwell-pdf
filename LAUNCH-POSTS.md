## Quick answers for common questions people ask in comments

- **"Is this open source?"** → Not currently — happy to talk licensing if
  you want to build on it, but the code isn't public.
- **"Does it work offline?"** → Yes, once the page has loaded once, since
  everything runs client-side.
- **"What about scanned/image-only PDFs?"** → It overlays text and marks on
  top of the page; it doesn't do OCR or recognize handwriting on a scan.
- **"Any plans for a mobile app?"** → It already works in mobile browsers —
  no separate app needed.
